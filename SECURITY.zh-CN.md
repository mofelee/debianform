## 支持的版本

DebianForm 目前处于公开测试阶段。安全修复面向最新的 beta 版本和 `main` 分支。

安全模型、仅限 root 的 SSH 边界、secret 处理、state/lock 行为及发布产物验证记录在：

https://github.com/mofelee/debianform/blob/main/docs/security-model.zh.md

## 报告漏洞

请通过本仓库的 GitHub Security Advisories 报告安全漏洞：

https://github.com/mofelee/debianform/security/advisories/new

请勿为疑似漏洞创建公开 issue。我们会尽快确认有效报告，并通过 advisory 讨论串协调修复。
