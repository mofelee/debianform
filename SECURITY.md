# Security Policy

## Supported Versions

DebianForm is currently in public beta. Security fixes target the latest beta
release and the `main` branch.

Security model, root-only SSH boundary, secret handling, state/lock behavior,
and release artifact verification are documented in:

https://github.com/mofelee/debianform/blob/main/docs/security-model.zh.md

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub Security Advisories for
this repository:

https://github.com/mofelee/debianform/security/advisories/new

Avoid opening public issues for suspected vulnerabilities. We will acknowledge
valid reports as quickly as possible and coordinate fixes through the advisory
thread.
